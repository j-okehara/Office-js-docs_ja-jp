
# <a name="defaultlocale-element"></a>DefaultLocale 要素
アドイン内の文字列で使用されるロケールの既定カルチャ名を指定します。

 **アドインの種類:**コンテンツ、作業ウィンドウ、メール


## <a name="syntax:"></a>構文:


```XML
<DefaultLocale>string </DefaultLocale>
```


## <a name="remarks"></a>注釈

en-US などの BCP 47 `language` タグの書式で値を指定します。
