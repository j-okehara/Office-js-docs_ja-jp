# <a name="icon-object-javascript-api-for-excel"></a>Icon オブジェクト (JavaScript API for Excel)

セルのアイコンを表します。

## <a name="properties"></a>プロパティ

| プロパティ       | 型    |説明| 要件セット|
|:---------------|:--------|:----------|:----|
|インデックス|int|指定したセット内のアイコンのインデックスを表します。|[1.2](../requirement-sets/excel-api-requirement-sets.md)|
|set|string|アイコンがその一部であるセットを表します。使用可能な値は次のとおりです。Invalid、ThreeArrows、ThreeArrowsGray、ThreeFlags、ThreeTrafficLights1、ThreeTrafficLights2、ThreeSigns、ThreeSymbols、ThreeSymbols2、FourArrows、FourArrowsGray、FourRedToBlack、FourRating、FourTrafficLights、FiveArrows、FiveArrowsGray、FiveRating、FiveQuarters、ThreeStars、ThreeTriangles、FiveBoxes。|[1.2](../requirement-sets/excel-api-requirement-sets.md)|

_プロパティのアクセスの[例を参照してください。](#property-access-examples)_
