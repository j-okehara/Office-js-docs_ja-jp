# <a name="inkstroke-object-(javascript-api-for-onenote)"></a>InkStroke オブジェクト (JavaScript API for OneNote)

_適用対象:OneNote Online_    


インクの 1 つのストロークを表します。

## <a name="properties"></a>プロパティ

| プロパティ     | 型   |説明|フィードバック|
|:---------------|:--------|:----------|:-------|
|id|string|InkStroke オブジェクトの ID を取得します。読み取り専用です。|[移動](https://github.com/OfficeDev/office-js-docs/issues/new?title=OneNote-inkStroke-id)|

_プロパティのアクセスの[例を参照してください。](#property-access-examples)_

| リレーションシップ | 型   |説明| フィードバック|
|:---------------|:--------|:----------|:-------|
|floatingInk|[FloatingInk](floatingink.md)|InkStroke オブジェクトの ID を取得します。読み取り専用です。|[移動](https://github.com/OfficeDev/office-js-docs/issues/new?title=OneNote-inkStroke-floatingInk)|

## <a name="methods"></a>メソッド

| メソッド           | 戻り値の型    |説明| フィードバック|

## <a name="method-details"></a>メソッドの詳細


### <a name="load(param:-object)"></a>load(param: object)
JavaScript レイヤーで作成されたプロキシ オブジェクトに、パラメーターで指定されているプロパティとオブジェクトの値を設定します。

#### <a name="syntax"></a>構文
```js
object.load(param);
```

#### <a name="parameters"></a>パラメーター
| パラメーター    | 型   |説明|
|:---------------|:--------|:----------|
|param|object|省略可能。パラメーターとリレーションシップ名を、区切られた文字列または 1 つの配列として受け入れます。あるいは、[loadOption](loadoption.md) オブジェクトを提供します。|

#### <a name="returns"></a>戻り値
void
